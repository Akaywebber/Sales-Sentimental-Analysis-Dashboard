# Sales Dashboard Project

## Overview
This repository contains a *Sales Dashboard* project designed to visualize and analyze product reviews. It includes the Power BI dashboard, the datasets used for analysis, and the associated Jupyter Notebook used for data cleaning and processing.

## Files in this Repository

| File Name | Description |
|-----------|-------------|
| SalesDashboard.pbix | The main Power BI dashboard file showing KPIs, charts, and slicers. |
| ProductReviews.csv | Dataset containing product review data including ratings, sentiments, and other metrics. |
| DataProcessing.ipynb | Jupyter Notebook used to clean and preprocess the data before importing into Power BI. |

## Features of the Dashboard
- *KPI Cards:* Show totals and averages for key metrics like review counts, average ratings, and sentiment scores.
- *Interactive Slicers:* Filter data by product, category, or sentiment.
- *Charts:* Line charts, bar charts, and pie charts to visualize trends and distributions.
- *User-friendly design:* Rounded edges, color-coded visuals, and a clean header for easy interpretation.

---

## How to Use
1. Clone or download this repository.  
2. Open SalesDashboard.pbix in *Power BI Desktop*.  
3. Use the slicers to interactively filter data.  
4. Explore KPIs and charts to analyze product review trends.

---

## Requirements
- *Power BI Desktop* (for opening .pbix file)  
- *Python 3.x* and Jupyter Notebook (optional, for exploring DataProcessing.ipynb)  
- Required Python libraries: pandas, numpy, matplotlib (for notebook)
